# honeybee_revive_standards/ — reference datasets

Reference Phius-REVIVE data shipped as JSON. Consumed by the model packages.

## Subfolders

- `appliances/` — appliance reference data.
- `cambium_factors/` — Cambium grid-emissions factors (`_generate_json_files.py` is a CPython dev script — pandas).
- `CO2_measures/`, `national_emission_factors/` — carbon/emissions reference data.
- `programtypes/`, `schedules/` — REVIVE-relevant programs and schedules.

## Notes
- Data is JSON, not model logic. Tests: `../tests/`.
