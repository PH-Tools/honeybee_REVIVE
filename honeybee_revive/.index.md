# honeybee_revive/ — core REVIVE model

Phius-REVIVE attributes on base Honeybee objects. Registered via `_extend_honeybee_revive.py`. See `../context/ARCHITECTURE.md`.

## Key modules (IPy2.7-safe — loads in Rhino)

- `_extend_honeybee_revive.py` — registers the REVIVE property slot (runs on import).
- `fuels.py`, `grid_region.py`, `national_emissions.py`, `CO2_measures.py` — carbon/emissions model objects.
- `properties/` — per-host-object property classes (own `to_dict()`/`from_dict()`).
- `cli/`, `__main__.py` — command-line entry.

## `output/` — CPython-only (pandas)

Resilience reporting: `resilience_hourly_data.py`, `resilience_summer_graphs.py`, `resilience_winter_graphs.py`, `_shared.py`. **Uses pandas — do not import from the Rhino load path.** See `../context/CODING_STANDARDS.md`.
